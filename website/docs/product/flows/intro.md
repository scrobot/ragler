# Primary User Flows

## Happy Paths

### Confluence → Chunking → Save (Support)
1. User supplies link.
2. System fetches and chunks.
3. User reviews in **Simple Mode**.
4. User saves to collection.

### Confluence → Chunking → Save (Developer)
1. User supplies link.
2. System fetches and chunks.
3. User reviews in **Advanced Mode** (split/merge available).
4. User saves.
