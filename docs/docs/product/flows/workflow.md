# Workflow Reference

## Flow 1: Ingest to Publish

1. Create collection.
2. Ingest source.
3. Generate and edit chunks.
4. Preview and publish.
5. Validate via search.

## Flow 2: Maintain Existing Collection

1. List collection chunks.
2. Edit/split/merge/reorder.
3. (Optional) Use collection agent suggestions.
4. Validate with search queries.

## Flow 3: Incident-style Validation

1. Identify missing/incorrect answer.
2. Locate relevant collection and chunk.
3. Update chunk content.
4. Re-run query to confirm fix.
