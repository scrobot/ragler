# Feature Technical Requirements (FTR)

Each FTR in this directory maps one feature area to implementation-ready requirements.

## Standard Structure

- Purpose
- Scope
- Functional requirements
- Non-functional constraints
- Acceptance checks
- Dependencies and risks
